# Politique de sauvegarde — Ma Laverie

## Objectif

Éviter la perte de données.

## Méthode

- Sauvegarde régulière de la base de données
- Sauvegarde du code sur GitHub

## Fréquence

- Sauvegarde automatique chaque jour (prévision)
- Sauvegarde manuelle après modification importante

## Stockage

- Données stockées sur serveur sécurisé
- Copie possible sur cloud

## Conclusion

La sauvegarde permet de sécuriser les données du projet.
