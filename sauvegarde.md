# Politique de sauvegarde — Ma Laverie

## Objectif

Éviter la perte de données et garantir la continuité du projet.



## Données concernées

- Base de données (laveries, machines, utilisateurs)
- Code source



## Méthodes de sauvegarde

- Sauvegarde automatique de la base de données
- Sauvegarde du code via GitHub



## Fréquence

- Sauvegarde quotidienne de la base de données
- Sauvegarde après chaque modification importante du code



## Stockage

- Données stockées sur serveur sécurisé
- Possibilité de sauvegarde sur cloud



## Restauration

- Possibilité de restaurer les données en cas de problème
- Conservation des anciennes versions



## Conclusion

La sauvegarde permet de protéger le projet contre les pertes de données.
