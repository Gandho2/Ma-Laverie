# Déploiement — Staging

## Objectif

Déployer l'application dans un environnement de test avant mise en production.

---

## Solution choisie

Pour ce projet, un déploiement simple est utilisé :

- exécution locale (localhost)
- possibilité de déploiement sur une plateforme gratuite (Render)


## Étapes

1. Installer les dépendances (npm install)
2. Lancer le serveur (node index.js)
3. Accéder à l'application via http://localhost:3000
4. Tester les routes avec Postman


## Limites

- Pas encore de serveur VPS
- Pas encore de base de données réelle
