## Comparaison des technologies backend

Plusieurs technologies étaient possibles pour le backend.

Node.js est une solution moderne qui permet d’utiliser JavaScript côté serveur. Il est assez simple à prendre en main et permet de développer rapidement.

PHP est également une solution simple et très utilisée, mais elle est moins moderne dans certains cas.

Java est une technologie très puissante et sécurisée, mais elle est plus difficile à apprendre et à mettre en place pour un projet de ce niveau.



## Justification du choix

J’ai choisi Node.js car il correspond mieux à mon niveau actuel.

Il me permet d’utiliser le même langage (JavaScript) pour le frontend et le backend, ce qui simplifie le développement.

De plus, il est adapté pour créer une API simple et rapide pour mon application.
