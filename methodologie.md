# Méthodologie de projet — Ma Laverie

## Choix de la méthodologie

Pour ce projet, j’ai choisi d’utiliser une méthode Agile.

## Justification

La méthode Agile permet de travailler progressivement et de s’adapter aux besoins du projet.

Elle est adaptée à mon projet car :
- le projet peut évoluer
- les fonctionnalités peuvent être ajoutées progressivement
- elle permet une meilleure organisation

## Organisation du projet

Le projet est organisé en plusieurs étapes :

- Analyse du besoin
- Rédaction du cahier des charges
- Mise en place technique (GitHub, Docker, CI)
- Développement du backend
- Améliorations futures

## Gestion des tâches

Les tâches sont organisées avec GitHub Projects sous forme de tableau :
- To do
- In progress
- Done
- backlog
## Conclusion

Cette méthode me permet de mieux organiser mon travail et de suivre l’avancement du projet.
