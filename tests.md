# Tests — Ma Laverie

## Objectif

Vérifier que la fonctionnalité de réservation fonctionne correctement.

---

## Tests unitaires

Les tests unitaires ont été réalisés avec Jest.

Ils permettent de tester la logique métier de la réservation.



## Cas testés

- Réservation réussie lorsque la machine est libre
- Réservation refusée lorsque la machine est occupée



## Résultat

Les tests ont été exécutés avec succès :

- 2 tests réalisés
- 2 tests validés

Cela confirme que la fonctionnalité fonctionne correctement.


## Outils utilisés

- Jest : framework de test
- Supertest : test des requêtes HTTP
